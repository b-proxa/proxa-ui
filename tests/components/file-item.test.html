Props / Classes Reference

Class Description
.file-item Base container. Flex row with gap, padding, hover state. Max-width: 300px.
.file-item__icon File type icon (36x36). Add type modifier for color.
.file-item__icon--xlsx Green background for Excel files.
.file-item__icon--pdf Red background for PDF files.
.file-item__icon--doc Blue background for Word files.
.file-item__icon--pptx Gold background for PowerPoint files.
.file-item__icon--csv Green background for CSV files.
.file-item__icon--default Gray background for unknown types.
.file-item__info Info container. Flex: 1, handles truncation.
.file-item__name Filename with ellipsis overflow.
.file-item__meta Secondary info (size, date).
.file-item__actions Action button container.
.file-item__btn Action button. Gray background.
.file-item__btn--delete Delete button. Red hover state.
.file-list Container for multiple file items.
.file-list-empty Empty state message.
.source-link Inline link variant with icon.
