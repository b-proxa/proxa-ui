Props / Classes Reference

Class Description
.hub-card Base container. Flex column layout with padding, border, hover state.
.hub-card__top Header row for icon and optional badge. Uses flex with space-between.
.hub-card__title Card title text. Semi-bold, slightly larger.
.hub-card__meta Secondary metadata text. Smaller, muted color.
.hub-grid 4-column responsive grid layout for hub cards.
