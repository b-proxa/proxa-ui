Props / Classes Reference

Class Description
.btn Base button class. Default transparent background with border.
.btn--primary Primary action button. Dark background, white text.
.btn--sm Small size variant. Reduced padding and font size.
.btn--lg Large size variant. Increased padding and font size.
.btn--icon Icon-only button with equal padding.
.loading Loading state. Shows spinner, hides text.
.btn-ai Mini AI trigger button.
.btn-add Mini add button (16x16).
