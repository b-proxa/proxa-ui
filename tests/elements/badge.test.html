Props / Classes Reference

Class Description
.badge Base badge class. Gray background, rounded corners.
.badge--success Green background. Use for completed, active, or positive states.
.badge--warning Yellow/gold background. Use for in-progress or attention-needed states.
.badge--error Red background. Use for errors, failed, or negative states.
.badge--outline Transparent with border. Use for subtle labels or categories.
.badge--count Pill-shaped notification count. Blue background, centered text, min-width enforced.
