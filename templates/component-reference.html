Assembly Tips

How to Assemble Pages
Step 1: Pick a template from the Templates section (Dashboard, Presentation, Report, Data Review)

Step 2: Copy component HTML from this reference

Step 3: Replace the "DROP ZONE" placeholders in the template with actual components

Step 4: Customize content (text, values, labels)

Step 5: Test on mobile (resize browser to check responsive behavior)
Required Imports

All pages need this CSS import: 

<link rel="stylesheet" href="../css/proxa.css">

For interactive components (dropdowns, panels, modals), add: 

<script type="module">
    import { initDropdowns } from '../js/dropdown.js';
    import { showToast } from '../js/toast.js';
    import { openPanel, closePanel } from '../js/slide-panel.js';

    initDropdowns();
</script>
