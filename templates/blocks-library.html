Blocks Library

Copy-paste HTML blocks for assembling pages. Each block is self-contained with responsive behavior built in. Use placeholders like {VALUE} for dynamic content.

Assembly Guide

How to Assemble Pages
Step 1: Pick a template from the Templates section (Dashboard, Presentation, Report, Data Review)

Step 2: Identify the ZONE comments in the template - these show where blocks go

Step 3: Copy block HTML from this library and paste into the appropriate zone

Step 4: Replace {PLACEHOLDER} values with actual content

Step 5: Test on mobile (resize browser to verify responsive behavior)
Template Zone Format
<!-- ZONE: metrics-row | BLOCKS: metric-card, metric-grid | REQUIRED: no --> <div class="metrics-row"> <!-- Drop metric-card or metric-grid blocks here --> </div> <!-- /ZONE: metrics-row -->
Required CSS Import:
<link rel="stylesheet" href="../css/proxa.css">
